# 🤖 בוט הסיכומים - הוראות הפעלה

## 📁 מיקום הבוט
הבוט נשמר בתיקיה: `בוט סיכומים` 

## 🚀 הפעלה מהירה
```bash
# כניסה לתיקיה
cd "בוט סיכומים"

# הפעלה אוטומטית
./auto-start.sh
```

## 📊 פקודות ניהול
```bash
# בדיקת סטטוס
pm2 status

# צפיה בלוגים
pm2 logs telegram-bot

# הפעלה מחדש
pm2 restart telegram-bot

# עצירה
pm2 stop telegram-bot

# מחיקת תהליך
pm2 delete telegram-bot
```

## 🔄 הפעלה אוטומטית מלאה
הבוט מוגדר לרוץ אוטומטית בהפעלה מחדש של השרת!

## 📝 הערות חשובות
- הבוט רץ ב-PM2 שמבטיח יציבות
- הטוקן של הבוט נשמר בקובץ index.js
- מסד הנתונים נשמר בקובץ data.db
- הבוט תומך בעברית מלאה

## 🛠️ פתרון בעיות
אם הבוט לא מתחיל:
```bash
# בדיקה שכל התלויות מותקנות
npm install

# בדיקה שPM2 מותקן
npm install -g pm2

# הפעלה ידנית
node index.js
```

💡 **טיפ**: פשוט תכנס לתיקיה והבוט יתחיל אוטומטית!
