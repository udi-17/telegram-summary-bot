## ⚡ התחלה מהירה (3 שלבים):

### 1️⃣ קבל טוקן
- פתח טלגרם → חפש `@BotFather`
- שלח `/newbot` → תן שם לבוט
- **שמור את הטוקן!**

### 2️⃣ התקן ספריות
```bash
pip install -r requirements.txt
```

### 3️⃣ הפעל!
- פתח `telegram_bot.py`
- החלף `"YOUR_BOT_TOKEN_HERE"` עם הטוקן שלך
- הפעל:
```bash
python telegram_bot.py
```
