# 🍕 בוט וואטסאפ לפיצריה

בוט וואטסאפ חכם למכירת מוצרי פיצריה עם ממשק ניהול מתקדם ועיצוב מודרני.

## ✨ תכונות הבוט

### 🤖 בוט וואטסאפ חכם
- הזמנת מוצרים דרך וואטסאפ
- תפריט דינמי עם קטגוריות
- עגלת קניות מתקדמת
- מעקב הזמנות בזמן אמת
- תמיכה מלאה בעברית

### 🎨 ממשק ניהול מתקדם
- דשבורד עם סטטיסטיקות
- ניהול הזמנות וסטטוסים
- ניהול מוצרים וקטגוריות
- ניהול לקוחות
- ניתוחים וסטטיסטיקות
- הגדרות מותאמות אישית

### 📊 ניהול נתונים
- מסד נתונים SQLite
- שמירת הזמנות ומוצרים
- היסטוריית לקוחות
- גיבוי אוטומטי

### 🔧 פונקציות מתקדמות
- עיבוד הודעות בזמן אמת
- ניהול סטטוסי הזמנות
- מערכת התראות
- דוחות וסטטיסטיקות

## 🚀 התקנה והפעלה

### דרישות מוקדמות
- Node.js גרסה 14 ומעלה
- npm או yarn
- טלפון עם וואטסאפ (לסריקת QR)

### שלבי התקנה

1. **התקנת dependencies:**
```bash
npm install
```

2. **הפעלת הבוט:**
```bash
npm start
```

3. **הפעלה במצב פיתוח (עם auto-reload):**
```bash
npm run dev
```

4. **גישה לממשק הניהול:**
פתחו את הדפדפן והיכנסו לכתובת:
```
http://localhost:3000
```

## 📱 התחברות לוואטסאפ

1. **הפעלת הבוט:**
   - הרצו את הפקודה `npm start`
   - הבוט יציג QR Code בטרמינל

2. **סריקת QR Code:**
   - פתחו וואטסאפ בטלפון
   - לכו ל"וואטסאפ Web"
   - סרקו את ה-QR Code שמופיע בטרמינל

3. **אישור התחברות:**
   - אשרו את ההתחברות בוואטסאפ
   - הבוט יהיה מוכן לשימוש

## 🍕 שימוש בבוט

### פקודות זמינות:
- `התחל` - התחלת שיחה
- `תפריט` - הצגת התפריט המלא
- `עגלה` - צפייה בעגלת הקניות
- `סיים` - סיום הזמנה
- `עזרה` - הצגת עזרה

### איך להזמין:
1. שלחו `תפריט` לראות את המוצרים
2. שלחו את שם המוצר (למשל: "פיצה מרגריטה")
3. שלחו `עגלה` לראות את ההזמנה
4. שלחו `סיים` לסיום ההזמנה
5. שלחו את כתובת המשלוח
